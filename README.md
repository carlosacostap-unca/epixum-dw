# Epixum - Diseño Web (DW)

Este proyecto es la base para el curso de Diseño Web. Originalmente basado en `epixum-mobile`, ha sido adaptado para el desarrollo web.

## Descripción

Epixum es una plataforma de gestión educativa. Esta versión (DW) se enfoca en las tecnologías y prácticas modernas de diseño y desarrollo web.

## Tecnologías

- [Next.js](https://nextjs.org)
- [React](https://react.dev)
- [Tailwind CSS](https://tailwindcss.com)
- [PocketBase](https://pocketbase.io) (Backend)

## Comenzando

Primero, ejecuta el servidor de desarrollo:

```bash
npm run dev
# o
yarn dev
# o
pnpm dev
# o
bun dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.
